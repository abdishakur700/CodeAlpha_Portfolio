# 🌐 Personal Portfolio Website

A modern, responsive personal portfolio website built using **HTML**, **CSS**, and **JavaScript**. This project showcases my skills, projects, resume, and contact information in a clean and interactive design.

## 📖 Overview

This portfolio serves as my online professional profile where visitors can:

- Learn more about me
- View my technical skills
- Explore my projects
- Download my resume
- Contact me through the contact section

The website is fully responsive and provides a smooth user experience across desktop, tablet, and mobile devices.

---

## ✨ Features

- 👤 About Me section
- 💻 Skills section
- 📂 Projects showcase
- 📄 Resume download
- 📞 Contact section
- 📱 Fully responsive design
- 🎨 Modern and clean UI
- ⚡ Smooth scrolling navigation
- ✨ Scroll animations
- 🖱️ Hover effects and transitions

- JavaScript


Simply open the `index.html` file in your web browser.
